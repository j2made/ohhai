Oh, Hai.

ohhai.js is a jQuery plugin that adds a class to an element when it becomes visible in the viewport.

View on Github

Install with bower:

bower install ohhai

Reference jQuery and ohhai.js:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>
<script src="ohhai.js"></script>

Initialize:

$('.on-visible').ohhai({
  // Default Options
  elTrigger: 'top',       // `top` or `bottom`, trigger point for element
  triggerOffset: 0,       // distance the trigger should be delayed.
  offsetValue: 'px',      // `px` or `percentage`, the value type for `triggerOffset`
  inViewClass: 'visible'  // class name that should be added
});
